\section{Startseite}
\label{home}

Wenn man die Webseite zum Ersten mal aufruft, landet man auf der Startseite (Home-view). 

\begin{figure}[H]
    \begin{center}
      \frame{\includegraphics[width=0.8\textwidth]{Website/Home-page.png}}
      \caption{Die Startseite}
    \end{center}
\end{figure}

Hier findet man unser \nameref{logo}, als sowohl auch ein Knopf wo man zu einer \nameref{map} kommt mit Überblick über alle 
Parks. Rechts vom Logo befindet sich eine \nameref{slideshow} welche immer zwischen die ersten Bilder 
aller Parks umschaltet. Drückt man auf die Slideshow kommt man auf die \nameref{parkDetails} 
Seite des Parks, dem das Bild gehört.
